# Data Mining and Databases
## Create venv

+ On Linux/MacOS
```
python3 -m venv .venv
```
+ On Windows
```
py -m venv .venv
```
Than activate 

+ On Linux/MacOS
```
.venv/bin/activate
```
+ On Windows
```
.venv\Scripts\activate
```
And finally install requirements
```
pip install -r requirements.txt
```
